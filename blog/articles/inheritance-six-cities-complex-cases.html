六都地政繼承疑難案例 2026:遺囑爭議、代位繼承、國外繼承人、寶璣實戰

「爸爸立的遺囑全部留給弟弟,我們其他人怎麼辦?」、「繼承人住在美國、大陸、加拿大,文件怎麼處理?」——**繼承的複雜情境遠超法條條文**,實戰才是關鍵。本文整合寶璣建設 10 年六都繼承實戰,涵蓋:遺囑爭議(自書/代筆/公證)、代位繼承、國外繼承人、跨國資產、百年家族等疑難情境,並深度解析六都特色,讓您精準理解自家繼承的可能挑戰。

遺囑爭議的完整處理

遺囑的 5 種法定方式

依《民法》第 1189 條:

1. 自書遺囑(親筆)

2. 公證遺囑(公證人)

3. 密封遺囑(公證人密封)

4. 代筆遺囑(3 人見證)

5. 口授遺囑(緊急情況)

各方式的要件(民法 1190-1195)

自書遺囑(最常見)

- 親自書寫全文。

- 註明年、月、日。

- 親自簽名。

- 若有塗改增刪,需註明並簽名。

- 無須見證人。

公證遺囑

- 公證人作成。

- 2 位見證人在場。

- 遺囑人口述、公證人筆記宣讀講解、遺囑人認可後記明年月日、公證人、見證人及遺囑人同行簽名。

- 最有公信力。

代筆遺囑

- 3 位見證人。

- 遺囑人口述、見證人之一筆記宣讀講解、經遺囑人認可後記明年月日及代筆人姓名,全部同行簽名。

常見遺囑爭議

爭議 1:真偽

- 自書遺囑的筆跡爭議。

- 代筆遺囑的見證人程序。

- 公證遺囑相對較無爭議。

爭議 2:遺囑能力

- 被繼承人立遺囑時是否有意識能力?

- 是否受脅迫、詐欺?

- 老人失智後的遺囑效力。

爭議 3:內容侵害特留分

- 全部留給一人 → 其他繼承人特留分被侵害。

- 可行使特留分扣減權。

爭議 4:見證人資格

- 依民法 1198 條,繼承人、受遺贈人不得為見證人。

- 否則遺囑無效。

爭議 5:遺囑是否仍有效

- 立遺囑後,被繼承人處分部分遺產 → 該部分遺囑失效。

- 立新遺囑與舊遺囑衝突 → 新遺囑優先。

遺囑真偽鑑定

筆跡鑑定:

- 委任專業鑑定機構。

- 比對被繼承人生前筆跡。

- 費用 5-20 萬。

文書鑑定:

- 紙張年代。

- 墨水種類。

- 印章真偽。

證人詢問:

- 代筆遺囑見證人。

- 公證人(若為公證遺囑)。

遺囑爭議訴訟

訴訟類型:

- 確認遺囑無效之訴。

- 特留分扣減訴訟。

- 分割遺產訴訟。

時程:1-3 年。

費用:律師費 50-200 萬。

代位繼承的實務案例

代位繼承複習

依民法 1140 條:

- 第 1 順位繼承人先於被繼承人過世或喪失繼承權。

- 其直系血親卑親屬代位繼承其應繼分。

實務複雜情境

情境 1:多代代位

- 祖父留遺產。

- 兒子(父親)先於祖父過世。

- 父親的兒子(你)代位繼承。

- 但你也先於祖父過世。

- 由你的兒子(曾孫)繼續代位。

- 可能多達 3 代代位。

情境 2:多位代位人

- 父親先過世,留 3 個子女。

- 祖父過世時,3 個孫子共同代位。

- 分配:父親原應繼分 ÷ 3。

情境 3:養子女代位

- 父親的養子女可代位(法律地位相同)。

- 但養父母的親生子女與養子女之間無法互相代位。

情境 4:代位 + 拋棄繼承

- 父親先過世,孫子本可代位。

- 但孫子拋棄代位繼承權。

- 孫子的子女(曾孫)不能再代位(民法第 1176 條例外)。

- 與一般繼承不同!

情境 5:代位繼承的遺產稅

- 代位繼承人繼承原被代位人的應繼分。

- 遺產稅依代位人數分別計算扣除。

- 多位代位人各有 56 萬扣除額。

實戰案例:3 代代位的複雜分配

家族結構:

- 曾祖父(2026 過世,被繼承人)。

- 祖父(2020 過世,先於曾祖父)。

- 父親(2025 過世,先於曾祖父,但在祖父之後)。

- 現代你 + 你哥(目前健在)。

分析:

- 祖父先於曾祖父 → 祖父的子女代位。

- 父親為祖父的子女,本可代位,但父親在曾祖父前已過世。

- 父親的子女(你 + 你哥)繼續代位。

- 你們兩兄弟代位祖父,再代位父親的那份。

應繼分:

- 假設曾祖父有 3 個子女(祖父 + 2 個大叔父)。

- 祖父份額 = 1/3。

- 父親代位祖父 = 1/3(若父親獨子)。

- 你 + 你哥再代位父親 = 1/3 ÷ 2 = 各 1/6。

這是寶璣處理過的真實案例

- 家族:台南後壁百年家族。

- 被繼承人:曾祖父(103 歲過世)。

- 涉及 4 代 27 位繼承人(含代位、拋棄、失聯者)。

- 處理時程:18 個月。

- 寶璣服務費:120 萬。

- 最終:祖產 8 甲農地順利繼承。

國外繼承人的處理 SOP

常見情境

- 繼承人移民美、加、澳、紐、歐、日。

- 繼承人在大陸工作、定居。

- 被繼承人有海外資產。

- 繼承人跨多國(美、台、陸)。

處理原則

原則 1:繼承權不分國籍

- 法定繼承人不論國籍、居住地,均有繼承權。

- 不能因為住國外就排除。

原則 2:適用台灣民法

- 被繼承人死亡時為台灣人(或在台有遺產)。

- 繼承依台灣法律。

原則 3:文件需驗證

- 海外文件必須經台灣機構驗證。

- 確保真實性。

海外繼承人文件準備(各國別)

美國、加拿大、澳洲、紐西蘭

授權書 / 同意書辦理:

- 步驟 1:在當地請公證人(Notary Public)公證簽名。

- 步驟 2:送州務卿(美)/ 省政府(加)/ 聯邦(澳紐) 認證。

- 步驟 3:送駐當地台北經濟文化辦事處驗證。

- 步驟 4:寄回台灣。

- 整體時程:1-3 個月。

- 費用:USD 200-800。

歐洲(英、法、德、荷等)

- 類似程序:公證人 → 外交部 → 駐歐辦事處。

- 時程:1-3 個月。

日本、韓國

- 日本:公證人 → 法務局 → 日本外務省 → 駐日辦事處。

- 韓國:公證 → 外交部 → 駐韓辦事處。

- 時程:1-2 個月。

中國大陸

特殊程序:

- 步驟 1:大陸公證處公證。

- 步驟 2:海峽兩岸關係協會(海協會)驗證。

- 步驟 3:送到台灣海峽交流基金會(海基會)認證。

- 步驟 4:在台使用。

- 時程:2-4 個月。

- 費用:人民幣 500-3,000。

香港、澳門

- 中國公證服務公司 → 海協會 → 海基會。

- 類似大陸流程。

必備的海外文件

1. 繼承系統表同意書:確認自己為繼承人。

2. 拋棄繼承聲請書(若選擇拋棄)。

3. 遺產分割協議書。

4. 委任書(委任律師、地政士)。

5. 印鑑證明(或相當於印鑑的簽名認證)。

簡化流程:視訊 + 電子簽名(新趨勢)

- 部分機關接受視訊確認身分。

- 電子簽名(Adobe、DocuSign 等)。

- 仍需後續原件寄回。

寶璣的全球網絡

- 合作律師:美、加、澳、英、日、中、香港。

- 認證流程:熟悉各國程序。

- 全程代辦:客戶僅需配合簽署。

- 費用:海外聯絡 + 認證,視國別 5-20 萬。

六都繼承特色分析

台北市

特色:

- 資產最龐大:豪宅、商辦、公司股權。

- 遺產稅最高:單案常達數千萬至數億。

- 複雜度:多位海外繼承人(子女移民率高)。

- 爭議多:前妻子女、遺囑真偽常見。

典型案例:

- 老企業家 10 億資產。

- 3 子女,2 位美國、1 位台灣。

- 遺囑爭議 + 特留分扣減。

- 時程 2-3 年。

新北市

特色:

- 中產家庭最多:透天、公寓 + 少量土地。

- 繼承人分散:移居六都各處。

- 土地 73-1 風險:老祖產未辦多。

典型案例:

- 三峽老透天 + 農地。

- 5-10 位繼承人。

- 老祖父、父親累代未辦。

- 時程 1-2 年。

桃園市

特色:

- 客家家族多:派下權、祭祀公業。

- 農地比例高:農用、節稅規劃。

- 航空城影響:土地升值潛力。

典型案例:

- 中壢、平鎮客家家族。

- 祭祀公業或累代農地。

- 時程 1.5-2.5 年。

台中市

特色:

- 科技新貴 + 傳統家族並存。

- 中科周邊土地:高增值。

- 海線、山線差異大。

典型案例:

- 沙鹿、梧棲傳統家族(海線)。

- 中科工程師家族(都心)。

- 時程 6 個月-2 年。

台南市

特色:

- 百年家族最多:清朝、日治祖產。

- 農業繼承多:稻米、果園。

- 累代繼承頻繁:多代未辦。

- 祭祀公業氣氛濃。

典型案例:

- 後壁、白河百年稻米家族。

- 跨 3-4 代未辦繼承。

- 30-50 位繼承人。

- 時程 1.5-3 年。

高雄市

特色:

- 美濃、旗山客家家族(重點)。

- 都心老家族(新興、前金)。

- 農業 + 工業混合。

典型案例:

- 美濃客家家族,祭祀公業派下權繼承。

- 家族 80-150 人。

- 時程 2-3 年。

六都共通的疑難情境

1. 累代繼承:所有都市都可能發生。

2. 海外繼承人:大都市更常見。

3. 遺囑爭議:高資產家庭多。

4. 特留分扣減:遺囑侵害時。

5. 73-1 條風險:農業縣市(桃園、台南、高雄)尤甚。

常見問題 FAQ

我爸立的遺囑全部給大哥,我和其他兄弟姊妹真的一分都沒有嗎?
**不是,您仍有特留分**。依民法 1223 條,您的特留分 = 應繼分 × 1/2。**實例**:(1)**家族有配偶 + 4 子女(含大哥)**:應繼分各 1/5;(2)**特留分**:1/5 × 1/2 = 1/10 per 人;(3)若總遺產 1 億,您應繼分 2,000 萬,**特留分為 1,000 萬**。**行動**:(1)**立即委任律師**;(2)**2 年內**提**特留分扣減訴訟**(知悉侵害時起);(3)向大哥或繼承遺產者**請求扣減**。時程:1-2 年。費用:50-200 萬律師費。**寶璣經驗**:100% 特留分訴訟都能爭取到應得份額,關鍵在**時效**與**專業律師**。
被繼承人立的遺囑是自書的,但我懷疑不是他寫的,怎麼辦?
**委任律師做筆跡鑑定 + 文書鑑定**。(1)**筆跡鑑定**:(a)委任專業鑑定機構(如中央警察大學、學術機構);(b)比對被繼承人生前筆跡(信件、合約等);(c)費用 5-20 萬;(d)時程 1-3 個月。(2)**文書鑑定**:(a)紙張年代(是否在立遺囑日期前生產);(b)墨水成分;(c)印章真偽。(3)**證人詢問**:同住家人、律師、醫師等。(4)**提確認遺囑無效之訴**:(a)訴請法院確認;(b)訴訟 1-3 年;(c)費用 80-300 萬。**實戰經驗**:若能證明遺囑為偽造,遺囑無效,**依法定繼承分配**。寶璣處理過多件遺囑偽造案,成功率 75%+。
繼承人住國外,要回台灣辦理嗎?費用多少?
**不用回台,可透過海外文件辦理**。**做法**:(1)**美加澳紐**:當地公證 → 駐外辦事處認證 → 寄回台灣,時程 1-3 個月,費用 USD 200-800;(2)**大陸、港澳**:公證處 → 海協會 → 海基會,時程 2-4 個月,費用人民幣 500-3,000;(3)**歐洲、日韓**:公證 → 外交部 → 駐當地辦事處,時程 1-3 個月。**必備文件**:(a)繼承權同意書;(b)遺產分割協議書;(c)委任書;(d)印鑑證明(或簽名認證)。**寶璣協助**:(1)全球合作律師網絡;(2)全程代辦各國認證;(3)費用 5-20 萬(依國別)。**關鍵**:越早通知海外繼承人越好,避免 6 個月時效壓力。
我家在台南有清朝時期的祖產,從來沒辦過繼承,現在能處理嗎?
**可以,但需專業累代繼承處理**。**挑戰**:(1)**跨 4-5 代繼承人**,可能 50-100 人;(2)**日治時期戶籍資料**(需特別調閱);(3)**原始文件考證**(鬮分字、合約字);(4)**30% 可能已進入 73-1 列管期**。**寶璣處理經驗**:(a)台南後壁 5 代繼承,28 個月完成;(b)費用 **80-500 萬**(依複雜度);(c)成功率 **80-90%**(視土地是否已歸國有)。**行動建議**:(1)**立即查土地狀態**:是否已列管、公告、標售;(2)**委任寶璣評估**:初次諮詢免費;(3)若可救援,**越早費用越低**。**30% 客戶選擇後續處分土地**,寶璣可直接收購,價格合理。
寶璣建設的繼承服務,跟一般代書、地政士有什麼不同?
**完整專業團隊 + 一條龍服務**。差異:(1)**一般代書/地政士**:(a)專精地政登記;(b)稅務、法律諮詢有限;(c)無歷史考證能力;(d)費用較低但範圍窄;(2)**寶璣建設**:(a)**律師 + 會計師 + 地政士 + 歷史考證**完整團隊;(b)處理簡單到極複雜案件;(c)**跨國繼承人**;(d)**累代繼承**;(e)**訴訟代理**;(f)後續不動產處分(可由寶璣收購)。**適合寶璣的情境**:(a)遺產 5,000 萬以上;(b)繼承人 5 人以上;(c)有爭議或複雜情境;(d)跨國、跨代。**適合單純地政士**:(a)單純小額繼承;(b)無爭議;(c)繼承人少。**寶璣承諾**:初次諮詢免費,若您的案件單純,寶璣會推薦合作地政士處理,**不勉強接單**。
